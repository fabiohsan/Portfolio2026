Seção 02 · Prioridade 01

Acessos
Técnicos.

Itens críticos. Sem eles, não é possível migrar o servidor, integrar os imóveis nem configurar o funil de leads.

Domínio
Registro.br — Contato Técnico
Para o apontamento DNS de unus.com.br ao novo servidor. Recomendamos a adição como "Contato Técnico" — sem necessidade de compartilhar login ou senha.
Como fazer
registro.br → Meu Registro → selecione o domínio unus.com.br → Contatos → Adicionar Contato Técnico → informe o ID: [SEU ID REGISTRO.BR]. Procedimento padrão e seguro, amplamente utilizado por agências.
Integração de Imóveis
VISTA API — Usuário + API Key
Credenciais para importarmos os 600 imóveis de forma automatizada e com atualização em tempo real.
Como encontrar
No painel VISTA: Configurações → Integrações → API Keys → Gerar Nova Chave. Copie o Usuário e a API Key e envie via WhatsApp.
CRM — Leads
C2S — Token de Acesso
Para configurarmos o recebimento de leads direto no funil do C2S, sem perda de nenhum contato gerado pelo novo site.
Como encontrar
No painel C2S: Perfil / Configurações → API → Token de Acesso → Copiar. Ou crie um usuário admin com o e-mail [SEU E-MAIL].
SEO & Blog
WordPress (Impacto) — Acesso Admin
Para resgatarmos o histórico de SEO, URLs indexadas e conteúdo do blog antes da migração — protegendo o ranqueamento no Google.
Como adicionar acesso
wp-admin → Usuários → Adicionar Novo → e-mail [SEU E-MAIL] → Papel: Administrador → Adicionar Usuário.

Seção 03

Identidade Visual
& Marca.

Essenciais para o padrão luxury. Materiais em baixa resolução ou formatos inadequados comprometem o resultado final.

Marca
Manual da Marca + Logotipos em Vetor
SVG, AI ou EPS — todas as versões (positivo, negativo, monocromático). Paleta com hexadecimais e CMYK.
Como enviar
Use Google Drive ou WeTransfer (wetransfer.com) para o envio. Não envie via WhatsApp — o app comprime arquivos e compromete qualidade dos vetores.
Banco de Imagens
Fotos da Equipe — Mín. 2 MP
Fotografias profissionais para as páginas institucionais. Resolução mínima: 2000 × 1200 px. Formatos: JPG ou PNG sem compressão.
⚠ Acima de 50 MB: use WeTransfer ou Google Drive. Não envie pelo WhatsApp — a compressão automática destrói a qualidade.
Vídeos Institucionais
Para o player do site (Bunny.net) ou YouTube. Formatos: MP4 (H.264), MOV ou link do YouTube/Vimeo. Qualidade mínima: 1080p Full HD.
Como enviar
Acima de 500 MB: Google Drive + compartilhe o link. Para vídeos publicados, envie apenas a URL do YouTube/Vimeo.
⚠ Não envie vídeos pelo WhatsApp — o app comprime e reduz resolução abaixo do mínimo necessário.
Referências Visuais
Benchmark — 2 ou 3 sites de referência
Links de portais imobiliários de luxo que você considera "campeões de vendas". São o norte visual exato do projeto — tipografia, hierarquia, cores e tom.
Exemplos de referência
Referências internacionais: sothebysrealty.com · christiesrealestate.com · compass.com. Qualquer portal que você admire — nacional ou internacional — serve como referência válida.
